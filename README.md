# **Pocket Utilities**
My first Minecraft mod!
## Features
- Pocket Crafting Table: A portable crafting table that can be used anywhere.
## Version/Requirements
- Fabric: `>= 0.18.4`
- Minecraft: `~1.20.1`
- Java: `>= 17`
- Fabric API: `any`
- Fabric Language Kotlin: `any`
